# Form-Backend-for-Learning

User Management CRUD App

This project is a simple User Management system that allows creating, reading, updating, deleting, and searching users.
The application uses a simulated class-based in-memory storage instead of a real database.

It was built to practice backend architecture, REST APIs, query parameters, and data filtering.

Tech Stack

Node.js, Express.js, JavaScript (ES6), In-memory class-based storage (simulated database).

This project demonstrates RESTful API design principles including CRUD operations, query-based filtering, and resource-based routing using Express.
